Ejemplos de sockets

Servidor de eco stream basado en select

Este ejemplo plantea proporcionar un servicio de eco a múltiples clientes usando un servidor secuencial (fichero eco_serv_select_tcp) que utiliza select para esperar simultáneamente por un evento de cualquier cliente.

El cliente es el mismo que se usó en el primer ejemplo (fichero eco_clie_tcp), que mantiene la conexión con el servidor durante todo su tiempo de vida.

Ejecute varios clientes para observar cómo el servidor va atendiendo sus peticiones de forma intercalada.
