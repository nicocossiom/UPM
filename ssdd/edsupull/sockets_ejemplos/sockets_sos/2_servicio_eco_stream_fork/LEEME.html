Ejemplos de sockets

Servidor de eco stream concurrente con fork

Este ejemplo plantea proporcionar un servicio de eco concurrente usando procesos convencionales (fork).

El cliente es el mismo que se usó en el primer ejemplo (fichero eco_clie_tcp), que mantiene la conexión con el servidor durante todo su tiempo de vida.

Ejecute varios clientes para observar cómo el servidor va atendiendo sus peticiones de forma concurrente.

Nótese cómo se ignora la señal SIGCLD para evitar que los procesos hijos de servicio se queden zombis.
