Ejemplos de sockets (IPv4)

Cliente HTTP usando funciones de E/S estándar de C

Este ejemplo muestra el uso de funciones de stdio para hacer transferencias a través de los sockets, construyendo para ello un cliente de HTTP (fichero cliente_http).

Como puede observarse en el código, usando fdopen se puede asociar un stream de stdio con un socket, permitiendo usar todas la funciones estándar de C para enviar y recibir datos por el socket. Nótese que, como se verá en un próximo ejemplo, estas funciones, además de ofrecer un mayor grado de abstracción, eliminan el problema de la recepción parcial de datos.

Puede probar este programa usando ejemplos como los siguientes: 

./cliente_http www.fi.upm.es 80 /
./cliente_http laurel.datsi.fi.upm.es  80 /~ssoo/
